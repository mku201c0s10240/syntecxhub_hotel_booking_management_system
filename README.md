A simple Java console-based hotel room booking system that allows users to view available rooms, book rooms, and cancel bookings.
This project is designed for beginners to understand Object-Oriented Programming, arrays, loops, and basic Java logic.

# Features
1. View all available rooms
2. Book a room by entering room number
4. Cancel an existing booking
5. Stores booking details including customer name
6. Interactive menu-driven program
7. Simple and clean console interface
8. Uses Class, Objects, Arrays, Methods, Loops, Scanner input
